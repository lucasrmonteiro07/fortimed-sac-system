🔧 COMO EXECUTAR O SQL NO SUPABASE
════════════════════════════════════════════════════════════════════════════════

1. Acesse o Supabase: https://supabase.com/dashboard
2. Selecione seu projeto: fortimed-sac-system
3. Vá para: SQL Editor (menu lateral)
4. Clique em "New query"
5. Cole o SQL abaixo:

════════════════════════════════════════════════════════════════════════════════

ALTER TABLE occurrences 
ADD COLUMN IF NOT EXISTS transportadora TEXT;

CREATE INDEX IF NOT EXISTS idx_occurrences_transportadora 
ON occurrences(transportadora);

════════════════════════════════════════════════════════════════════════════════

6. Clique em "Run" ou pressione Ctrl+Enter
7. Pronto! ✅
