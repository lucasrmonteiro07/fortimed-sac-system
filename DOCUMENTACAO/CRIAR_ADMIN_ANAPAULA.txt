### OPÇÃO 2: SQL Direto (Supabase Console)
──────────────────────────────────────────

1. Acesse: https://app.supabase.com/
2. Vá em: SQL Editor
3. Execute o comando:

   INSERT INTO public.users (email, name, role, created_at, updated_at)
   VALUES ('anapaula@fortimeddistribuidora.com.br', 'Ana Paula', 'admin', NOW(), NOW())
   ON CONFLICT (email) DO UPDATE SET 
       name = 'Ana Paula', 
       role = 'admin', 
       updated_at = NOW()

4. Clique em "Run"
5. Novo usuário criado!
