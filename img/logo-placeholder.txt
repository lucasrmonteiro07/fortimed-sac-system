# LOGO DA FORTIMED

## 📁 Instruções para adicionar o logo:

1. **Adicione o arquivo logo.png nesta pasta:**
   - Caminho: `img/logo.png`
   - Formato: PNG
   - Tamanho recomendado: 200x80 pixels
   - Fundo transparente (opcional)

2. **Se não tiver o logo:**
   - O sistema usará o emoji 🏥 como fallback
   - Ou você pode usar qualquer imagem PNG

3. **Teste:**
   - Abra o sistema no navegador
   - Verifique se o logo aparece no header
   - Verifique se aparece na página de login

## 🎨 Especificações técnicas:

- **Altura no header**: 50px
- **Altura no login**: 60px
- **Formato**: PNG (recomendado) ou JPG
- **Fundo**: Transparente ou branco
- **Qualidade**: Alta resolução para diferentes telas
